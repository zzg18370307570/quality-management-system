# 工业质量管理系统

本项目是一个基于Python Flask开发的工业质量管理系统，作为本人的毕业设计作品。

## 项目概述

系统主要功能包括：
- 用户认证与授权管理
- 产品质量数据追踪
- 质量问题管理与处理
- 数据分析与报表生成

## 技术栈
- 后端：Python Flask
- 前端：HTML/CSS/JavaScript
- 数据库：MySQL
- 短信服务：阿里云SMS
